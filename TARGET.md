## 目前无效的快捷键
- Ctrl + Shift + Z(撤回)，无效，目前只有 Ctrl + Y 有效
- Alt + 上下箭头(把当前行文本上下移动) 无效，目前只能对自己新输入的文字有效，和光标位置无关，已存在的行无效
- Alt + Shift + 上下箭头(把当前行文本上下复制) 无效，目前只能对自己新输入的文字有效，和光标位置无关，已存在的行无效
- Ctrl + Enter (新开一行，不把文本后面内容移动到下一行) 无效，它还是和 Enter 一样的行为
- Ctrl + Shift + Enter (在上方新开一行)无效
- Ctrl + Tab/Ctrl + 1-9 (切换文件缓冲区) 无效，我打开多个文件，按下后没反应
- Ctrl + S (保存文件) 无效，提示 "File exists (add ! to override)"
