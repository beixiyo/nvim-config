## 🎯 快速扩展选择

Vim 提供了多种快速扩展选择范围的方法：

**基础操作：**
- `v` - 进入字符可视模式（逐字符选择）
- `V` - 进入行可视模式（逐行选择）
- `<C-v>` - 进入块可视模式（矩形选择）
- `gv` - 重新选择上次的选择范围

**文本对象（Text Objects）扩展选择：**

在可视模式下，使用文本对象可以快速扩展选择范围：

| 操作 | 功能 | 示例 |
|------|------|------|
| `iw` / `aw` | 选择单词（inner word / a word） | `viw` 选择当前单词 |
| `i"` / `a"` | 选择引号内容（inner quotes / a quotes） | `vi"` 选择引号内的内容 |
| `i'` / `a'` | 选择单引号内容 | `vi'` 选择单引号内的内容 |
| `i(` / `a(` 或 `ib` / `ab` | 选择括号内容 | `vi(` 选择括号内的内容 |
| `i{` / `a{` 或 `iB` / `aB` | 选择花括号内容 | `vi{` 选择花括号内的内容 |
| `i[` / `a[` | 选择方括号内容 | `vi[` 选择方括号内的内容 |
| `it` / `at` | 选择标签内容（HTML/XML） | `vit` 选择标签内的内容 |
| `ip` / `ap` | 选择段落 | `vip` 选择整个段落 |
| `is` / `as` | 选择句子 | `vis` 选择整个句子 |

**说明：**
- `i` 前缀表示 "inner"（内部），只选择内容，不包括分隔符
- `a` 前缀表示 "around"（周围），包括分隔符一起选择
- 在可视模式下，重复按相同的文本对象可以扩展选择范围

**实用技巧：**
1. **逐步扩展**：先按 `v` 进入可视模式，然后按 `iw` 选择单词，再按 `ip` 扩展到段落
2. **切换端点**：在可视模式下按 `o` 可以切换选择的起始/结束端点
3. **重新选择**：使用 `gv` 可以重新选择上次的选择范围
4. **配合移动命令**：在可视模式下使用 `w`、`e`、`b`、`}`、`)` 等移动命令可以扩展选择

**示例场景：**
```vim
" 场景1：选择函数参数
function test(a, b, c)  " 光标在 a 上
vi(                    " 选择括号内的所有参数

" 场景2：选择整个函数
function test() {      " 光标在函数内任意位置
vi{                    " 选择花括号内的内容（函数体）

" 场景3：选择引号内容
const str = "hello"    " 光标在引号内
vi"                    " 选择 "hello"（不包括引号）
va"                    " 选择 "hello"（包括引号）
```
